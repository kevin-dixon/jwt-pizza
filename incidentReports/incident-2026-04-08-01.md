# Incident: 2026-04-08 11-34-00

## Summary

```
Between the times of 11:30 and 11:45 on April 8, 2026, all users encountered failed purchases. The event was triggered by external chaos testing at 11:34. The chaos testing contained an automatic pizza creation failure in the pizza factory.

This chaos code caused pizza creation to fail, resulting in all pizza purchases to fail. The event was detected by Grafana logging and monitoring triggered by a drop in pizza creations and an increase in error logs. The team started working on the event by inspecting logs and researching the root of the purchase failures. This critical incident affected 100% of users.
```
